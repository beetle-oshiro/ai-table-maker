# AI Table Maker

## 概要

AI（ChatGPT API）を活用して、データベースのテーブル作成を補助するWebアプリです。
ユーザーが入力した内容をもとにAIが整理し、PostgreSQLにテーブルを作成します。

---

## 使用技術

- Python（Flask）
- PostgreSQL
- OpenAI API（ChatGPT）
- HTML

---

## 主な機能

- テーブル作成（AIによる整理付き）
- テーブル一覧表示
- データ登録機能
- データ一覧表示

---

## 工夫した点

- AIを「司令塔」として使い、SQLはPython側で安全に生成
- 初心者でも扱いやすいシンプルなUI
- ブラウザ上で完結する操作
